1. PowerPoint Automation<br/><br/>
Link: https://esther-t.github.io/SideQuestsLab/<br/><br/>
Problem: My relative who is an elderly is always asking me to help turn their photos into PowerPoint presentations for work. They'd send me the images over WhatsApp, I'd manually create the file, paste each photo in, resize and rotate them one by one, then send it back. Simple enough but when it happens regularly, it really adds up.<br/><br/>
Solution: So I built SlideBuilder using free version Claude and Cursor, and hosted for free on GitHub pages. This is a tool where they can just drop in their photos and get a ready-to-send PowerPoint in seconds. No more middleman, no more back and forth. It auto-corrects photo orientation, fits every image neatly to the slide, and lets them reorder or rotate anything before downloading. Just open it in a browser and it works. Built for free, saves time for everyone.
